# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## v0.1.0 [2026-03-10]

### Added

- Documentation: installation instructions, usage guides, and API examples.
- EuroMillionsCombination & EuroDreamsCombination: classes for generating and validating draws for those games.
- LotteryCombination: core class for creating and managing lottery combinations, including rank evaluation.
- Combination: utility class providing helpers for combination creation and rank handling.
- BoundCombination: wrapper class offering convenience methods for bounded/partial combinations.
- Ranking: implementation of combination rank calculation logic.
- Initial commit: project scaffold and first implementation of core features.
